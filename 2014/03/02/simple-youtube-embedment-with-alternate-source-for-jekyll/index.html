Simple YouTube Embedment With Alternate Source for Jekyll

As this blog is potentially for viewers from Mainland China, it’s necessary to have a backup video for the stupid GFW. This plugin can use YouTube and Tudou as video sources.

Installation: Put youtubeCN.js in ROOT_PATH/source/javascripts and youtube_gfw.rb in ROOT_PATH/plugins.
