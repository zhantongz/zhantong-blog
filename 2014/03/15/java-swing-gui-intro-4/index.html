Java Swing GUI Intro (4)

We’ve talked about flow-layout, an easy-to-use layout. However, sometimes, you’d like let the user change the window size while maintaining the relative layout. That’s when we should use grid-layout. UPDATING

Grid-layout is a layout (obviously) that make a grid on the window. You can specify the number of rows and columns and put components into a box of the grid.
